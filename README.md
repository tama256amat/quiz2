# Features

ユーザーが自分と家族の身を守ることにつながる Quiz を学べます。

# Usage

1. index 画面下の「クイズを解いてみる」を押す
   2．問題を読み、正解だと思う選択肢「A」「B」「C」を押す
2. 解説ページ下の「次のクイズを解く」を押す

# Note

クイズは全部で 5 問あります。
